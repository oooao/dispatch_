# 派工整合系統

## 摘要
-----------
派工整合系統主要為媒合傢俱師傅與相關需求的客戶。透過線上預約的方式指派師傅完成傢俱安裝或修理，並整合相關其他服務如：電商、物流、特約等集合而成的平台。

透過Figma與設計師協調並協助APP UI設計
